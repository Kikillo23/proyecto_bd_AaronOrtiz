Para correr el proyecto: 

1. Se debe tener isntalado node, yo use la version 16.13.1
2. Abrir la terminal, en el directorio del proyecto
3. Ejecutar, npm install
4. Ejecutar, npm run build
5. Ejecutar, npm run start
